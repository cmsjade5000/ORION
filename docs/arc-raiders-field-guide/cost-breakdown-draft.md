# Arc Raiders Field Guide - Cost Breakdown Draft

## Assumptions
- Small personal project.
- Target budget: under $1,000 USD.
- Digital-first delivery.
- Focus on one-time costs; minimize monthly recurring expenses.
- Scope is for a single, comprehensive guide, not a series or a platform.

## Working Budget Table

| Category | Budget (USD) | Budget Type | Notes |
|---|---:|---|---|
| Tools and Software | 50 | Hard Cap | This is a strict hard cap. Reliance is on free/open-source tools. Any deviation requires explicit approval and justification; exceeding this cap would necessitate a significant re-evaluation of the project scope or require additional, unbudgeted funding. |
| Art and Design | 350 | Flexible Range | This budget allows for moderate AI image generation credits and sourcing free/low-cost assets. Flexibility means usage can be adjusted; more credits can be used if quality is paramount, or less if more free assets are found. However, premium AI models or assets require strong justification and may draw from Contingency only if strictly necessary and approved. |
| Writing and Editing | 200 | Hard Cap | A strict hard cap reflecting a DIY approach. The allocation is for essential external proofreading services only, ensuring a baseline quality without significant investment. Any additional editing services would be outside the scope of this budget. |
| QA and Review | 50 | Flexible Range | This range allows for dynamic allocation. If robust free feedback channels (e.g., community input) are sufficient, less of this budget is used. If specialized QA tools or a small number of external testers are deemed necessary, this budget can be utilized. This budget is not intended for extensive paid QA services; such needs would require a budget review. |
| Publishing and Distribution | 100 | Hard Cap | This is a firm hard cap based on predictable costs for domain registration (~$20/year) and static site hosting/CDN (e.g., GitHub Pages, Netlify - estimated ~$80 for the first year). Any increases in service costs would require absorbing them within this cap or seeking additional funding. |
| Resource Acquisition | 150 | Hard Cap | This is a strict limit for essential, licensed assets, specific research tools, or game-specific asset packs. Acquisitions beyond this limit are not permitted within the standard budget and would require a formal re-evaluation of project scope or funding. |
| Contingency | 100 | Contingency | This is a true contingency fund, reserved strictly for unforeseen critical needs or unavoidable overruns in hard-capped categories, and always requires review and explicit approval before use. It is not a buffer for flexible range categories or for non-essential upgrades. |
| Total | 1000 |  | Target cap. |
